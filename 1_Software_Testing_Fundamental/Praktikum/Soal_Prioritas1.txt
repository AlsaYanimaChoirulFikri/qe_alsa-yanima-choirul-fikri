# Soal Prioritas 1

1. Sebutkan dan jelaskan berbagai automation testing tool yang dapat digunakan (minimal 3)!
Jawaban :  
    - Appium : Appium biasanya digunakan untuk otomatisasi pengujian aplikasi seluler di platform Android
    - Postman : Postman sebelumnya digunakan untuk uji API dan sekarang sudah dapat berkembang untuk uji lainnya
    - Selenium : Selenium merupaka alat open-source yang biasanya digunakan pengujian web.

2. Sebutkan dan jelaskan urgensi / pentingnya proses pengujian pada perangkat lunak! (minimal 3 poin)
Jawaban : 
    - Menjamin kualitas produk : Pengujian yang bertujuan untuk membantu memastikan bahwa Software telah memenuhi standar kualitas sesuai dengan standar.
    - Menghindari kerugian finansial : Mendeteksi dan memperbaiki masalah pada tahap awal pengembangan, proses pengujian membantu mengurangi kemungkinan biaya yang terkait dengan perbaikan yang dapat menjadi mahal.
    - Memastikan pelanggan puasn: Software yang telah melalui pengujian menyediakan pengalaman yang lebih baik bagi para konsumen.
